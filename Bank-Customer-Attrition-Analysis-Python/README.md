# Bank Customer Attrition Analysis – Python

## Project Overview
Analyzed banking customer data to identify factors influencing customer churn and product usage behavior.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Credit card ownership vs churn
- Loan ownership vs churn
- Product usage analysis
- Credit score segmentation
- Tenure vs product usage
- Customer attrition trends

## Business Insights
- Customers using fewer products showed higher churn probability.
- Credit score and tenure impacted customer retention.
- Product engagement influenced churn behavior significantly.
